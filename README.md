# react-native-piwik-pro-sdk

React Native wrapper for Piwik Pro SDK

## Installation

```sh
npm install react-native-piwik-pro-sdk
```

## Usage

```js
import PiwikProSdk from "react-native-piwik-pro-sdk";

// ...

const result = await PiwikProSdk.multiply(3, 7);
```
